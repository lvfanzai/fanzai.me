Visual Studio 2015 中使用 gtest

@凡仔 · Aug 3, 2017 · 1 min read

Win 10 Visual Studio 2015 中使用 gtest 参考文献： http://www.top5solutions.com/google-test-visual-studio-configuration/

1. 新建 Win32 控制台项目工程，并去掉预编译头文件

2. 右键项目名，属性，设置gtest目录，包含gtest目录和include目录

3. 在工程源文件夹中添加gtest_main.cc和gtest-all.cc

4. 新建testcase.cpp文件，创建testcase

#include <limits.h> 
#include "gtest/gtest.h" 
#include "libalgo.h" 
class MultiplyTest : public testing::Test { 
};
TEST_F(MultiplyTest, findMean)
{
       int a; 
       a = print_test();
       EXPECT_EQ(0, a);
}

5. 编译成功，开始执行(不调试)即可出现测试结果
